系统管理

top

实时显示系统进程与资源使用情况。

df

显示磁盘信息

ps

列出所有进程信息
ps -ef|grep mysql

ps aux|grep pid 显示对应进程的详细信息。

jps

列出所有java进程信息
等价于ps aux|grep java

chmod

更改文件或目录权限

chown

更改文件或目录的所有者和群组

文件管理

ls\cd\mkdir\rm\rmdir\cp\mv

pwd

显示当前目录路径

touch

创建空文件

日志查看

vi/vim

cat

查看日志内容，显示所有内容

more/less

分页查看文件内容。less可以翻页，more不能翻页。查看较大的文本文件。

tail/head

tail -n 10

grep

查找

其他

ping

lsof

查看端口号信息
lsof -i:7051
